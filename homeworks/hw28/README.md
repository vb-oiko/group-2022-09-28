# generateList

Написати функцію generateList(array), яка приймає масив із чисел та масивів чисел і генерує список з елементів:

Приклад 1. [1, 2, 3]

<ul>
    <li>1</li>
    <li>2</li>
    <li>3</li>
</ul>

Приклад 2. [1,2, [1, 2, 3], 3, [5, 6, [8]]]

<ul>
    <li>1</li>
    <li>2</li>
    <ul>
        <li>2.1</li>
        <li>2.2</li>
        <li>2.3</li>
    </ul>
    <li>3</li>
    <ul>
        <li>3.5</li>
        <li>3.6</li>
        <ul>
            <li>3.6.8</li>
        </ul>
    </ul>
</ul>
